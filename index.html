Automated Deployment of Static Web Application

using GitHub Actions

A DevOps College Project by Team Techies

Project Description

This project demonstrates the automated deployment of a static web application using GitHub Actions. It implements a CI/CD pipeline to automatically build and deploy the website whenever changes are pushed to the repository. The website showcases team details, roles, and contributions.

Tools & Technologies

  • Frontend: HTML, CSS
  • Version Control: Git & GitHub
  • CI/CD: GitHub Actions
  • Hosting: GitHub Pages
